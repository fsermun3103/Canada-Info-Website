# Canada-Info-Website
Sitio web de información sobre Canada como práctica de HTML, CSS y Javascript con diseño responsive y dinámico.

## Páginas disponibles
#### HOME

Landing page con varias secciones:

- HERO: Incluye una imagen llamativa, un slogan potente y un botón para favorecer la interactividad.
- WHY-VISIT: Cuatro cards con información sobre por qué visitar Canadá.
- Experiences: 3 sub-secciones para que explicar el tipo de actividades que puedes realizar en este país.
- BEST-TIME: Cuándo es el mejor momento para tí para visitar Canadá y qué puedes encontrar en cada estación del año. Sección dinámica gracias a Javascript.
- CULTURE: Sección informativa sobre la historia, geografía, deporte nacional y fauna de Canadá, con elementos interactivos dinámicos con Javascript.

### Destinations

Página para explicar los destinos más populares en Canadá, que incluye:

- TOP-CITIES: Sección con las 3 ciudades más populares para el turismo, con stats personalizados para cada ciudad y explicación de que puedes hacer allí.
- WONDERS: Las maravillas del país. Esta sección se basa en un grid con las mejores rutas naturales de Canadá, pensado para hacer un galería de fotografías/vídeos.
- UNIQUE-EXPERIENCES: 4 cards que te dicen el tipo de experiencia que puedes seleccionar (WINTER, ADVENTURE, TRADITIONAL o NATURE) según tus preferencias, las fechas recomendadas y los lugares donde hacer estas actividades y botón de más información.

### ROUTES

Página explicativa de las rutas más interesantes y populares de Canadá:

- RUTA-DESTACADA: La primera ruta, y la que he considerado más interesante. Esta es la ruta que más he desarrollado, poniendo diferentes tags, una ruta con varias paradas, una descripción y que manera es la más sencilla para hacerla.
- RUTAS-RECOMENDADAS: Otras dos rutas que también están muy interesantes y que incluyen el número de días, lo medios de transporte recomendados y 3 paradas en ciudades.
- EXCLUSIVE-EXPEDITION: Con un formato parecido a las rutas anteriores, vemos esta ruta exclusiva, ya que es recomendada hacerla en un periodo de tiempo concreto, por situaciones meteorológicas.

### CONTACT

Página de contacto del sitio web:

- FORMULARIO: Aquí introudces tus datos para mandar un correo al administrador con un mensaje/pregunta.
- MAPA: Sección con un mapa interactivo con localizaciones de distintas Agencias de Turismo de Canadá, introducido mediante Javascript con Json.
- HOW-TO-GET-HERE: Sección de 3 cards que indican los mejores aeropuertos, conexiones globales e información para sacar el pasaporte y documentación necesaria.
